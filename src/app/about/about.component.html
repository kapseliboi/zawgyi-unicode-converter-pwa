About

Zawgyi Unicode Converter သည် ဇော်ဂျီ နှင့် မြန်မာ ယူနီကုဒ် တို့ကို အပြန်အလှန် ပြောင်းပေးနိုင်တဲ့ Progressive Web Application တစ်ခုပါ။ ၎င်းကို DagonMetric Myanmar Tools အဖွဲ့ကနေ ဖန်တီးခဲ့ပြီး အခမဲ့၊ open source အနေနှင့် မျှဝေထားခြင်းဖြစ်ပါတယ်။
